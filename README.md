# دورة بايثون للمرحلة الثانوية

هذه الدورة موجهة لطلاب المرحلة الثانوية الذين يرغبون في تعلم لغة البرمجة بايثون. ستساعدك الدورة على فهم المفاهيم الأساسية ف�� بايثون وكيفية استخدامها في حل المشكلات البرمجية.

## المحتوى
- **مقدمة في بايثون**: ما هي بايثون؟ ولماذا هي لغة شائعة؟
- **المتغيرات والبيانات**: كيفية العمل مع المتغيرات وأنواع البيانات.
- **هياكل التحكم**: التفرع (if statements) والتكرار (loops).
- **دوال بايثون**: كيفية تعريف واستخدام الدوال.
- **المكتبات والأدوات**: استخدام المكتبات المفيدة في بايثون.
